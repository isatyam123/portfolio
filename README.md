## ⚙️ Running Locally

Clone the repository:

```bash
git clone https://github.com/isatyam123/portfolio.git
```

Move to the project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
